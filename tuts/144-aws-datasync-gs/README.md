# Aws Datasync Gs

A read-only script that queries Datasync resources and displays information.

## Running

```bash
bash aws-datasync-gs.sh
```

## What it does

1. Listing agents
2. Listing locations
3. Listing tasks

## Resources created

None — this script is read-only.

## Cost

No cost. This script only reads existing resources.
